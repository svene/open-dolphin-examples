With Netbeans:
1. open project (pom.xml) in NetBeans
2. Run Project 'odwebjee'
3. open browser at: http://localhost:8080/od_web_02_remote_mode_jee/
4. click button 'make string longer on server'
5. text of label below button show: 'longer longer longer...' (one more 'longer ' each time you click the button)
